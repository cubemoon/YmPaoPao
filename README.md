#YmPaoPao 
基于Bmob服务。用户注册、登录、发送泡泡、评论、粉丝、关注、图片瀑布流、定位等功能。 练手项目

##数据
基于[Bmob](http://www.bmob.cn/)服务平台管理数据。

##状态
之前是通过开源中国的代码仓库来管理的。https://git.oschina.net/ymloveu/YmPaoPao
